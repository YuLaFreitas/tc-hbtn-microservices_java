O que é Spring Boot Framework?
É um framework que disponibiliza diversas ferramentas e facilita as configurações para desenvolvimento web, microserviços e aplicações stand alone.

Quais são os benefícios e recursos que Spring Boot fornece?
Fácil configuração, leve, possui servidor embarcado, gerenciador de dependencias.

Como é feita a adição de novas dependências (JPA, Hibernate, etc…) ?
Através do arquivo pom.xml

Na sua opinião, qual melhor IDE para utilizar para trabalhar como Spring Boot: Spring Tools Suites, IntelliJ IDEA, Eclipse ou NetBeans ?
O STS é simplificado e ideal para quem utiliza o eclipse, porém prefiro o intellij para desenvolver com Spring, por familiaridade com a IDE

O que é Spring Initializer?
É um site web para criação de projetos Spring
