O que é Spring Boot Framework?

 > O Spring Boot Framework cria aplicações estand alone (estand-alone), configura e cria a infraestrutura dos projetos.
> E, disponibiliza diversas ferramentas e facilita as configurações para desenvolvimento web, microserviços e aplicações stand alone.

Quais são os benefícios e recursos que Spring Boot fornece?

 Spring Boot Framework:
> Favilita as configurações.
> Tem um servidor embarcado (no caso Tomcat).
> Organiza o pom.xml
> Pelo dev tour reiniciamos o que fazemos, desabilitando o cache.
> Integra o live loaded, permitindo atualização automática.
> Realiza métricas, pelo actualtor.
> gerenciador de dependencias.

Como é feita a adição de (JPA, Hibernate, etc…) ?

>Através do arquivo pom.xml

Na sua opinião, qual melhor IDE para utilizar para trabalhar como Spring Boot: Spring Tools Suites, IntelliJ IDEA, Eclipse ou NetBeans ?

Spring Tools Suites.
